# 頑張れこうかとん
## 実行環境の必要条件
* python >= 3.10
* pygame >= 2.1

## ゲームの概要
主人公キャラクターこうかとんが地球を守るために敵と戦う

## ゲームの実装
###共通基本機能
* 授業内で実装した機能
### 担当追加機能
* 爆弾をこうかとんが被弾したときに泣き顔に画像を変えてしばらく動けなくする
* こうかとんが動けないときに爆弾が地球(画面下)に落ちるようにする
